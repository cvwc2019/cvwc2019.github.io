Overview

  • Detection: Datset in Pascal VOC format. Submission in COCO detection format
  • Pose: Dataset and submission in COCO format.
  • Plain ReID: Dataset constains cropped images with ID annotation. Submission should be a json like:

    • [
     0:[29,38,10,.......],
     3:[95,18,20,.......],
     ...
    ]
    where the number is image id, and the arrays are arranged in order from most similar to least similar.

  • ReID in Wild: Dataset only contains test set, which is the same as test set of detection track.

    • Submission should be a json of following schema:

    {
     "bboxs":[bbox],
     "reid_result":[
      id:[id]
     ]
    }

    bbox{
     "id": int, #used in reid_result
     "imageid": int,
     "pos": [x,y,w,h] #x,y is the left-top coord, all in pixel.
    }
    where the reid_result almost the same as in Plain ReID, with id replaced with bbox id.
